jQuery ajax 请求成功但不执行回调函数

在 Chrome 里边看 Network 网络请求，显示请求成功返回状态码 200 ，但是后续两个函数都执行，看文档还没能发现错误，Google 一搜索 ajax 请求后回调函数不执行，有一种情况就是设置从服务器返回的预期的数据类型不对，然后一去看我设置了两个回调函数，jQuery 的 ajax 请求语法是没有设置错误情况的回调函数的，去掉后边一个毁掉函数就能执行了。

iTerm2 快捷键

  1. 光标操作
    • ctrl + a: 到行首
    • ctrl + e: 行末
    • ctrl + r: 搜索命令历史，这个大家都应该很熟悉了
    • ctrl + k: 删除到文本末尾
    • ctrl + u: 删除一行
    • ctrl + d: 删除当前字符
    • ctrl + w: 删除光标前的单词
    • Command + r:清屏(其实是滚到新的一屏，并没有清空)
  2. 标签页操作
    • Command + T: 新建标签页
    • Command + W: 关闭标签页
    • Command + 左方向键: 前一个标签页( Shift + Command + [ )
    • Command + 右方向键: 后一个标签页( Shitf + Command + ] )
    • Command + 标签页编号: 进入标签页1，2，3…
  3. 面板操作
    • Command + D: 垂直分割
    • Command + Shift + D: 水平分割
    • Command + [: 前一个面板
    • Command + ]: 后一个面板
    • Command + Option + 上下左右方向键: 切换到上/下/左/右面板
  4. 文本选取
    • 鼠标双击选单词
    • 鼠标三击选整行
    • 选取某一部分，按住Shift，再点击某处，可以选取整个矩形内的文本
    • Command + D: 调出搜索面板，按Tab可以将选取部分向右扩展，按Shift + Tab向左扩展，按回车转到下一个匹配位置
  5. 其他
    • Command + Shift + h: 可以呼出复制历史
    • 鼠标选中某个路径或者某个词汇就自动复制
    • Command + Shift + Enter: 面板全屏
