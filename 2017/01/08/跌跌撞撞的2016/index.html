跌跌撞撞的2016

Jan 8, 2017 | Hits

似乎一直以来我都是过得很没有仪式感，像这种跨年或者是其他比较特殊的时刻，朋友圈微博总能看到很多人的感慨，我却十分的无感，总觉得这不就是普通的一天，日子不还是得过。并且拖延症来很严重，元旦3天假期就想要写总结了，心是这么想，但是行动却是严重的落后，一点一点拖竟然能让2号3号整整两天的时间就那么过去了，这很不应该。现在就下定了决心，立马行动，马上总结，让自己也有点仪式感。

去年这个时候还在准备期末考试，考到二十号才结束，考完试马上就和女朋友去桂林玩了，刚刚遇上寒潮，十分的冷，短短两三天的旅行却是非常不一般，有挺多的回忆与纪念。在出发之前，还和女朋友妈妈一起吃了个饭，见个面，也是人生时刻之一。桂林回来很快就回家了，那个时候我已经计划要学前端了，但是对前端的概念还是非常的模糊，工作内容、流程等等全都没有很清晰的概念。寒假回家带了 css3 秘笈，原计划是在寒假看完 W3C 的 html 文档以及这本 css3 秘笈，结果只是看了 w3cschool.com.cn 上的 html 教程而已，心是上进的，行动却是十足的小人，寒假几乎没有收获。搁到现在那点东西真的半天多就能看完的。

寒假结束，新学期开学，情感上折腾了会，然后去参加了前端圈大会，回来之后就彻底确定要转行自学了，学校的课程开始大规模的翘掉了，平均下来一个星期顶多上六七节课，成天泡在图书馆自己看书。纯粹的半路出家，对编程对前端半点概念都没，无头苍蝇一样到处乱碰乱撞。3月中旬百度有前端技术学院线上培训，一开始以为自己会也能够按照课程学下去，然后掌握前端基本技能。做了几天之后，发现自己根本跟不上进度，太多的东西要从头学了，最基本的 html 标签都是没掌握的。css3 秘籍也只是开学很潦草地看。自己想着把全部的课程都要做了，压根跟不上。那时大公司实习春招开始了，360最早，而且是伴随着一个线下训练营一起的，先完成一个线上作业，完成后根据评分再去参加线下培训，然后表现优秀的才能留下来暑期实习。啥都没想就报了名，笔试也压根没几道题会的，做到一半关了浏览器退出。可没想到居然过了线上笔试，严重怀疑是没有筛选的。百度前端技术学院受挫之后，计划着扎扎实实看 css3 秘籍，然后 js 高级教程的，360的通知一下子又让我掉头去试着完成线上第二轮任务了，用 js 写一个万年历，那个时候对 js 连一点皮毛的了解都说不上，怎么可能做得出来，看别人写好的 js 万年历源码都压根看不懂。折腾几天又放弃，接下来东看西看学了阮一峰的部分js教程，然后是很看的过了一遍 js 语言精粹。那个时候心里还是痴想着自己秋招还有机会进大公司，可实际上对前端连基本的工作流程工作内容还是不了解。不断地重新制定计划，就痴想着自己还有机会进大公司，一会想学这个，一会又去学那个，4月5月无头苍蝇乱撞一样就过了。

6月份突然发生了一件非常意料之外的事，急性气胸，其实4月份左右已经出现过一次轻微一点的症状了，6月初那次非常的急，自己只是在剧烈一点的运动或者是挤压到肺部的时候才会稍微胸闷的感觉，去省中医拍胸透一检查马上确诊为严重气胸，必要住院治疗。转院到了广医一院呼吸疾病研究所，查出病因是肺大泡破裂引起的。不做手术复发几率很高，做了手术如果还一直这么瘦依然很有可能复发，可是我现在还是那么瘦，fuck！自己了解了一些信息之后选择进行了肺大泡切除微创手术，没想到才二十二岁就做了人生的第一场手术。广医一院呼研所的医术毫无疑问是非常厉害的，单孔微创，隔天就能出院了，但是出院之后至少还有近一个月的时间是很难受的。这个经历很痛苦，非常可怕。很深刻地体会到了身体的重要性。

做完手术回来学校已经是6月中下旬，接下来又是期末考试，一个月多一点时间吧，压根没怎么学习，只是浏览了一遍锋利的 jQuery 这本书。期末考结束后，秋招早已经开始了，我掌握的东西还是依然非常的少，但我还是抱着一丝希望的。后来在前端相关的微信公众号里看到网易前端微专业的宣传介绍，然后很快的就报了名，暑假考完试开始马上就投入微专业的学习了，通过微专业基本上才在自己的脑海里形成前端开发的一个大概轮廓。但是由于基础的问题，我还是没能按照预期般的完成微专业的学习。因为发现自己继续学下去简历上也东西，没实战经验，秋招肯定没机会。然后就又半路放弃，开始想着做一个实时公交查询 web 应用，找同学要了接口，开始了实践。前前后后花了半个多月的时间吧，用 nodejs 的 Express 框架搭了个后台，解决了跨域和隐藏接口／密钥的问题。9月初的时候把应用部署到了腾讯云上。接下来两周就是硬着头皮去参加好几个大公司的秋季校招在线笔试，完成的情况都很糟糕，结果便是一个回音都没，完全没有面试机会。二十几号的时候，带着那个实时公交查询的作品投了简历给钱鹿，很幸运地过了。国庆后入职，实习三个月。

确定去钱鹿实习之后，我就放弃所有的校招了，知道自己的水平是够不着的。踏踏实实的实习上班打基础吧。
实习的前几周压力非常的大，真的没实战经验，计算机的基础也没。Vuejs框架之前都没接触过的，一下子就要上手使用。公司压根就不会再对你有什么培训，直接就要你上手开发任务了。在另外一篇《阶段工作总结》那记录些具体的东西。做了一个多月之后就慢慢上手了，后来由于后端排期的问题，后台项目就没有继续开发下去了，接着就接触了h5广告业务开发，一份转手了很多次的代码，很糟糕的代码。其实总共就做了两个的活动吧，圣诞活动以及新排行榜，基本上连 js 业务代码都没有怎么涉及到，只是写切页面写样式而已。自己在下班之余也没有很好地静下心去学东西，在前端技能的成长上是微乎其微的。现在回想起来，大一点的收获应该是：实际接触到了真实的开发环境，实战上遇到的问题也不再那么害怕，出现什么就会找方法去解决。更重要的是会快速去实践学习了，需要用到什么在比较短的时间内就去掌握。总结起来3点就是：快速学习、解决问题、了解真实开发。以前我从知道一个东西，到真正去学习并做出点东西来，总是要很长的准备时间。现在总算是体会到了什么是快速实践学习掌握。

在实习的最后一个月开始，我开始有点疲倦了，对于前端切页面有点反感，觉得这似乎不是自己想要的。现在看来也并不是这样，这些应该算是入门时的磨练。但这感觉却导致我产生了新的念头。实习结束前两周，也就是2016年最后几天，实习公司团队老大、hr先后找了我跟我聊了公司的决定，他们希望我能留下来，明年继续实习，7月份直接转为正式员工，待遇6*16。在说待遇的时候，hr一句后端的待遇通常是要比前端高的，因为后端技术含量更高。前半句毫无疑问是事实，后半句虽然我并不是很赞同，但在大多时候确实是这样。真正要掌握前端，技术含量不比后端低，或者这种比较是没有意义的。原本是一句很事实的话，并没有什么明显的错误，但与那段时间的疲倦以及对前途的焦虑，慢慢地就被放大了。首先，这个待遇实在是太低，一个月到可能连5k都没有，这样我就没法在今年贷款买车了。公司能给可能真的就这么多，创业公司实在拿不出更多的薪酬了；其次，公司的氛围我并不喜欢，同事的关系并没有我期待中那么的紧密，技术氛围也很一般，交流沟通都太少了，反正是很难成为朋友的，只会是同事；最后，目前我的前端技能还是很差的，依然有非常多的东西需要学，真正精通掌握并拿到自己满意的薪酬还是需要有非常多的东西要学，从去年决定学前端开始，我一直以来都觉得自己不会一直从事前端开发，自己的目标也就是5年内成为优秀的前端工程师，而5年之后我自己是不确定的。这个时候，新的念头、新的疑问就不由自主地冒出来了。

为什么我不直接选一个自己确定能长时间从事的职业？做5年前端开发，然后去转行去做其他的不是很浪费自己的经验积累？如果要选一个新的方向，那什么才是我有信心长时间从事下去的职业／行业呢？想着想着脑子里就冒出了自己一直有兴趣的领域，汽车！未来几年非常确定可以遇见的是，汽车一定会发生大变革。自己会选前端其实一定程度上自己心里也存在着畏惧，觉得自己不是计算机科班出身的，后端的难度太大了，自己会驾驭不了，更别说自动驾驶／人工智能这么高科技的前沿领域。但是现在自己也清楚了真正精通前端或者称为优秀的前端开发者难度也不小，需要付出的努力一样非常的多，成为优秀开发者之后再转行也很浪费经验，而且不确定性很大。因为害怕难度而选择看起来容易一点的领域，又想着以后转行，这是非常不对。深度学习／机器学习／人工智能上手的难度、门槛确实高了很多，但是我有信心在这个领域混十年二十年甚至十一辈子，那我怕什么难度，脚踏实地一点点去干呗。因为担心难度而南辕北辙是非常愚蠢的。在我大致确定着方向之后，alphago 升级版 master 在围棋对战平台上连胜全部的人类高手，再一次掀起了人工智能关注热点。我相信这里的热门与前端的热门是不一样的，十年后还存不存在前端开发这个职业还不一定，而人工智能十年后一定是大有作为的。并且深度学习／机器学习／人工智能这个领域的入门门槛是很高的，不像客户端前端开发一样参加个培训班就能上手入门。我确信我要从事这个领域。

其他大事：有了车，买了MacBook。
