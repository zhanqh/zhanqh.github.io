计算机基础

CPU 构架

  • 复杂指令集构架的 CPU 被称作 X86 构架的原因：Intel 的第一颗 CPU 的代号为 8086，后来依此架构又开发出80286, 80386…， 因此这种架构的CPU就被称为x86架构了。+
  • 64位的个人电脑 CPU 又称作 x86_64 构架

速度单位

  • Mbps 是 Mbits per second，故 100 Mbps = 12.5 Mbit/s

CentOS 7

  • 查看版本：cat /etc/redhat-release

设置 CentOS Minimal 模式虚拟机的分辨率大小

参考

  • 编辑文件 /etc/default/grub，修改 GRUB_CMDLINE_LINUX 字段，在其值末尾添加 vga=0x33F
  • 使用 grub2-mkconfig 来重新生成 /boot/grub2/grub.cfg 文件：grub2-mkconfig -o /boot/grub2/grub.cfg
  • 重启：reboot

配置网卡

参考

进入网卡配置目录
cd /etc/sysconfig/network-scripts
编辑配置文件：vi ifcfg-enp0s5，设置自动获取 IP，将ONBOOT=no修改为ONBOOT=yes
重启网卡

更改阿里云的 yum 源

参考

  • 备份原来的 yum 源
    mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.bak
  • 设置阿里云的 yum 源
    wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
