一般指令模式

光标移动

j 向下移动一个字符
k 向上移动一个字符
h 向左移动一个字符
l 向右移动一个字符
[n]j 向下移动 n 个字符
Ctrl + f 屏幕向下移动一页
Ctrl + b 屏幕向上移动一页
Ctrl + d 屏幕向下移动半页
Ctrl + u 屏幕向上移动半页

  • 光标移动到非空白字符的下一列
  • 光标移动到非空白字符的上一列
    [n] 向右移动 n 个字符
    0 移到行首
    ^ 移到行首第一个非空白字符
    $ 移到行末
    H 光标移动到屏幕第一行第一个字符
    M 光标移动到屏幕中间那一行第一个字符
    L 光标移动到屏幕最后一行第一个字符
    G 光标移动到文件最后一列
    gg 光标移动到文件第一列
    [n]gg 光标移动到文件第 n 列（nG 亦可）
    [n] 光标向下移动 n 列

搜寻与取代

/word 向下搜索关键词 word
?word 向上搜索关键词 word
n 重复上一个搜索动作
N 反向重复上一个搜索动作
:1,30s/word/WORD/g 在 1 至 30 行中用 WORD 字符替换 word
:1,$s/word/WORD/g 整个文件从头至尾使用 WORD 替换 word
:1,$s/word/WORD/gc 在交互式环境中，用 WROD 替换 word

+ g 显示行号信息
:1,30d 删除 1 至 30行

删除、复制与粘贴

x 删除光标所在字符
X 删除光标前一个字符
[n]x 从光标所在字符开始删除 n 个字符
[n]X 删除光标前 n 个字符
dd 删除光标所在那一整列
[n]dd 删除光标所在行开始向下的
d1gg 删除光标所在行到第一行的全部数据（d1G 亦可）
dG 删除光标所在行到最后一行的全部数据
d$ 删除光标所在字符到行末的全部字符
d0 删除光标前到行首的全部字符
yy 复制光标所在行
[n]yy 删除光标所在行开始向下 n 行
y1gg 复制光标所在行到第一行的全部数据（y1G 亦可）
yG 复制光标所在行到最后一行的全部数据
y0 复制光标所在前一个字符直行首的全部字符
y$ 复制光标所在字符直行末的全部字符
p 将已复制的数据粘贴到光标后
P 将已复制的数据粘贴到光标前
J 将光标所在行与下一行合成一行
c 重复删除多个数据，如重复向下删除 10 列，10cj
u 撤销
Ctrl + r 重做
. 重复前一个动作

一般指令模式切换到插入模式

插入模式、取代模式

i 光标所在处前插入
I 光标所在行第一个非空字符处插入
a 光标所在处后插入
A 光标所在行最后一个非空字符处插入
o 光标所在行下一行插入一个新行
O 光标所在行上一行插入新行
r 取代模式，取代光标所在字符一次
R 保持取代模式，一直取代直到退出

插入模式切换到命令行模式

储存、离开指令

:w 将编辑数据存入硬盘中
:w! 文件属性为只读时，强制存入（最终是否能存入与权限有关）
:q 退出 Vim
:q! 强制退出，不存盘
:wq 存盘后退出
ZZ 若文件修改过则存盘后退出，否则直接退出
:w filename 将编辑数据存储到新文件中
:r filename 打开新的文件将数据加载到光标后
:n1,n2 w filename 将 n1 到 n2 的内容储存成新文件
:! [command] 短暂离开 Vim 到命令行界面下执行 command 并显示结果

Vim 环境变更

:set nu 显示行号
:set nonu 取消显示行号

其他

区块选择

v 选中光标经过地方
V 选中光标经过行
[Ctrl]+v 区块选择，选中矩形框
y 复制选中地方
d 删除选中地方
x 删除选中地方
p 粘贴刚刚复制、删除的数据

多文件编辑

vim hosts /etc/hosts 同时打开多个文件
:files 列出打开的全部文件
:n 编辑下一个文件
:N 编辑上一个文件

多窗口编辑

:sp [filename] 打开新窗口，如果有 filename 则打开这个文件，否则打开同一文件
:[Ctrl]+w+j 向下切换窗口
:[Ctrl]+w+k 向上切换窗口
