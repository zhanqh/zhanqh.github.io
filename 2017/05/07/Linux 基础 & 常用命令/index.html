常用指令

  • exit 退出账号
  • command [-options] parameter1 parameter2 …
  • locale 查询当前语系
  • LANG=zh_CN.UTF-8 临时修改语系为简体中文
  • date 显示日期
  • date +%Y/%m/%d 格式化输出年月日
  • date +%H:%M:%S 格式化输出时分秒
  • cal 显示日历
  • cal 3 1994 指定年月，月缺省显示整年
  • man date 显示 date 指令的使用说明，man 为 manual 的缩写
  • man -f man 等价于whatis man
  • man -k man 等价于apropos man，搜索包含 man 关键字的指令
  • sync 数据同步写入磁盘
  • shutdown 关机
  • reboot 重启
  • chgrp [-R] 群组名称 文件或目录 更改文件群组，-R 参数为递归设置
  • chown [-R] 账号名称:群组名称 文件或目录 更改文件拥有者
  • chmod 改变权限，数字类型，r、w、x 权限分数分别为 4、2、1，每个权限的分数需要累加，如：-rwxrwx— 的数字值为 770
  • chmod 400 ssh 将 ssh 文件的权限设置为 -r——–
  • chmod u=rwx go=rx ssh 将 ssh 的权限设置为 -rwxr-xr-x
元件 内容 迭代物件 r w x
文件 详细数据data 文件数据夹 读到文件内容 修改文件内容 执行文件内容
目录 文件名 可分类抽屉 读到文件名 修改文件名 进入该目录的权限（key）
  • exit 退出当前账户
  • useradd [name] 新建用户
  • passwd 修改密码
  • su 进入 root 账户
  • root 账户下 passwd [name] 强制修改该账户的密码
  • 文件种类：-正规文件、d目录、l链接文件
  • echo $PATH 显示环境变量，$ 表示其后接的是一个变量
  • PATH="${PATH}:/usr/local/python3/bin"
  • mkdir 新建空目录
  • rmdir 删除空目录
  • cd 切换目录
  • touch 新建文件
  • pwd 显示路径
  • cp
  • mv
  • rm
  • cat 从第一行开始显示文件内容
  • tac 逆序，从最后一行开始显示
  • nl 显示文件内容，带行号
  • more 一页一页的显示内容
  • less 与 more 相似，还能向前翻页
  • head [-n number] 只显示头几行
  • tail [-n number] 只显示尾几行
  • od 二进制形式读取文件
  • umask 设置文件默认权限
  • file [filename] 查询文件类型
  • which [command] 可执行文件位置查找
  • whereis
  • locate [-ir] keyword 在/var/lib/mlocate/数据库中搜索
  • updatedb 更新上述数据库
  • find [PATH] [option] [action]
  • find / -mtime 0 24小时内更改过的文件
  • find / -mtime 3 3天前24小时内有更改过的文件
  • find /etc -newer /etc/passwd /etc 目录下比 passwd 文件更新的文件
  • find /home -user zhan /home 目录下用户 zhan 的文件
  • find / -name passwd 找出名为 passwd 的文件
  • find / -name "*passwd*" 找出名字包含 passwd 关键字的文件
  • find / -size +1M 找出系统中体积大于 1M 的文件
  • find / -name passwd -exec ls -l {} \; 将找到的文件列出其树状结构
  • df -h 将系统内所有的 filesystem 以易读的容量格式显示出来
  • ls [-sf] sourcefile targetfile 创建链接，不再任何参数为 hard link，加 -s 参数为 symbolic link，-f 参数表示强制链接
  • lsblk 列出系统上的所有磁盘列表
  • parted /dev/vda print 列出 vda 磁盘相关数据
  • partprobe -s 更新分区表信息
  • echo $PATH | cut -d ':' -f 5 取出 PATH 变量第 5 个路径值
  • export | cut -c 12- 取出 export 输出信息第 12 个字符之后的全部字符
  • last | grep 'root' 取出登陆信息中包含有 root 用户的行
  • last | grep -v 'root' | cut -d ' ' -f 1 不包含 root 的行的第一个分隔值
  • [alt] + [F1] 切换终端窗口
  • ps -ef | grep 'mongo' 查询 MongoDB 进程
  • kill -1 [PID] kill -SIGHUP [PID] 启动进程
  • kill -9 [PID] kill -SIGKILL [PID] 结束进程
  • kill -15 [PID] kill -SIGSTOP [PID] 暂停进程
  • tar -jcvf [filename] 压缩文件
  • tar -jtvf [filename] 查询压缩文件
  • tar -jxvf [filename] 解压文件
  • cat [filename] | less cat 结合 less 实现一页页查看文件
  • time ps aux time 命令测算 ps aux 指令执行所需要的时间
  • netstat -an | grep '80' 查询 80 端口监听情况
  • namp localhost 扫描 TCP 协议端口开启情况

目录

根目录

/bin 放置可执行文件
/boot 开机相关
/dev 设备
/etc 配置
/lib 库
/media 软盘、光盘、DVD
/mnt 挂载设备
/sbin 开机、网络服务指令
/srv 网络服务
/tmp 临时
/usr
/var
/home 使用主文件夹
/root 管理员主文件夹

/usr目录 Unix Software Resource

/usr/bin /bin 目录即链接至此
/usr/lib /lib 目录即链接至此
/usr/local 自行安装软件即位于此
/usr/sbin /sbin 目录即链接至此
/usr/share 主要放置只读架构的数据文件，包括共享文件

/var目录 Variable 与系统运行有关

/var/cache 存盘
/var/lib 程序执行本身所需要用的数据存放的文件
/var/lock 链接到/run/lock
/var/log/ 日志
/var/mail 个人电子邮件信箱
/var/run 程序或者目录启动后放置PID，链接到/run

shell

bash

  • type [-pa] <command> 查询命令是否为 bash 内置命令
  • Ctrl + u 删除整行命令
  • Ctrl + k 删除光标至行尾字符
  • Ctrl + a 光标跳到行首
  • Ctrl + e 光标跳到行末
  • Ctrl + w 删除光标至单词头
  • Ctrl + r 搜索命令，再按一次搜索下一条，按 上一条
  • echo ${PATH} 取变量 PATH 的值
  • alias lm='ls -al | more' 设置命令别名
  • unlias lm 取消命令别名
  • cmd ; cmd 连续执行多个命令
  • cmd1 && cmd2 第一个命令正确执行完毕后执行第二个命令，否则不执行第二个命令
  • cmd1 || cmd2 第一个命令执行完毕且出错时执行第二个命令，否则不执行第二个命令
