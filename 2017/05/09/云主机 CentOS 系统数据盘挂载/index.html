云主机 CentOS 系统数据盘挂载

May 9, 2017 | 实践 | Hits
文章目录

参考

  • 参看数据盘
    fdisk -l

  • 对数据盘进行分区
    fdisk /dev/vdb

  • 格式化新建分区
    mkfs.ext4 /dev/vdb1

  • 挂载新分区

执行 ls 命令显示 lost+found，则挂载成功

  • 查询文件系统
    df -h

  • 设置开机自动挂载
    echo '/dev/vdb1 /hdd ext4 defaults 0 0' >> /etc/fstab
