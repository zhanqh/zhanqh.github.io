requests

pip3 install requests

selenium

操纵浏览器 JS
pip3 install selenium

chromedriver

由于尚未安装 chromedriver 驱动，上述最后一个命令会报错。在官网下载 chromedriver 驱动放置到任一个系统环境变量 PATH 目录中，如：/usr/local/bin

phantomjs

brew install phantomjs

beautifulsoup4

pip3 install beautifulsoup4

from bs4 import BeautifulSoup

pyquery

语法与 jQuery 类似的解析库
pip3 install pyquery

pymysql

pip3 install pymysql

mysql.server start
mysql.server stop

pymongo

pip3 install pymongo

redis

pip3 install redis

flask

web 库，IP 代理等需要用到
pip3 install flask

django

web 服务器框架，分布式爬虫维护、后台管理系统
pip3 install django

jupyter

网页端记事本，调试
pip3 install jupyter

启动
jupyter notebook
