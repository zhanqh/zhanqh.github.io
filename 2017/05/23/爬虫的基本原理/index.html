爬虫：请求网站并提取数据的自动化程序

基本流程

  1. 发起请求（HTTP，Requests，headers）
  2. 获取响应（Response，HTML、JSON、二进制）
  3. 解析内容
  4. 保存数据

Requset

  • 请求方式
    • GET
    • POST
  • 请求地址 Request URL
  • 请求头 Request Header
    • User-Agent
    • Accept-Encoding
    • cookie
    • Host
  • 请求体 Form Data

Response

  1. 响应状态 Status Code
  2. 响应头 Response Headers（内容类型、内容长度、服务器信息，设置 cookie）

  3. 响应体 请求资源的内容




* response.text  响应文本
* response.content  响应二进制内容

抓取的数据类型

  1. 网页文本（HTML，JSON）
  2. 图片
  3. 视频
  4. 其他

解析方式

  1. 直接处理
  2. JSON 解析
  3. 正则表达式
  4. BeatufulSoup
  5. PyQuery
  6. XPath

解决 JS 渲染问题

  1. 分析 AJAX 请求
  2. Selenium/WebDriver
  3. Splash
  4. PyV8、Ghost.py

##保存数据

  1. 文本
  2. 关系型数据库
  3. 非关系型数据库
  4. 二进制文件
