Scrapy 的命令行工具

Jun 5, 2017 | 爬虫 | Hits
文章目录

全局命令：
startproject
genspider
settings
runspider
shell
fetch
view
version

项目命令：
crawl
check
list
edit
parse
bench

-h获取命令帮助信息
scrapy <command> -h

startproject创建项目
scrapy startproject myproject [project_dir]

genspider生成 spider
scrapy genspider mydomain mydomain.com
列出全部模板
scrapy genspider -l
指定模板
scrapy genspider -t crawl scrapyorg scrapy.org

crawl开启爬虫
scrapy crawl <spider>

check语法检查
scrapy check

list列出项目中的全部 spider
scrapy list

fetch利用 scrapy 下载获取网页
scrapy fetch <url>
–nolog 不输出日志
–headers 输出响应头信息
–no-redirect 禁止重定向

view下载保存页面，并在浏览器中打开
scrapy view <url>

shell命令行调试
scrapy shell <url>

parse测试回调方法
scrapy parse <url> [options]
scrapy parse <url> -c parse
scrapy parse https://weibo.cn/comment/F6SBLraNd\?uid\=5748579586\&rl\=1\#cmtfrm -c parse_detail --pipelines

settings获取配置信息
scrapy settings [options]
scrapy settings --get BOT_NAME
scrapy settings --get DOWNLOAD_DELAY
scrapy settings --get DOWNLOADER_MIDDLEWARES_BASE

runspider不创建项目直接运行爬虫文件
scrapy runspider <spider_file.py>

version查看依赖库版本
scrapy version -v
