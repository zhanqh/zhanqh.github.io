关于反爬虫

Jun 16, 2017 | 爬虫 | Hits
文章目录

传统反爬虫：

* IP 统计
* session 统计
* userAgent 判断
* 以上组合

新花样：

* JavaScript 做复杂请求逻辑，加 key/token
* JavaScript 混淆，变量名用阿拉伯语
* 在前端玩花样，如用图片、特殊字体渲染关键信息
* 频繁发布、改版
* 浏览器检测，如浏览器不同特性、Canvas 画图指纹

爬虫与反爬虫遵循经济学原理，存在边际效益，使对方付出更高的成本
误伤率、维护成本，点到为止
