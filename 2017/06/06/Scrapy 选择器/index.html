Scrapy 的选择器

Jun 6, 2017 | 爬虫 | Hits
文章目录
  1. 1. CSS
  2. 2. xpath

CSS

  • 文本
    response.css('title::text').extract_first()
  • 属性
    response.css('img::attr(src)').extract()
  • 条件
    response.css('a[href*=image]::attr(href)').extract()
    response.css('a[href*=image] img::attr(src)').extract()
  • 结合正则提取字符串
    response.css('a::text').re('Name\:(.*)')
    response.css('a::text').re_first('Name\:(.*)')

xpath

  • 文本
    response.xpath('//title/text()').extract_first()
  • 属性
    response.xpath('//img/@src').extract()
    response.xpath('//img[@src]').extract()
  • 条件
    response.xpath('//a[contains(@href, "image")]/@href').extract()
    response.xpath('//a[contains(@href, "image")]/img/@src').extract()
    response.xpath('//a[contains(text(), "点击")]/img/@src').extract()
    response.xpath('., "点击")]/img/@src').extract()
  • 正则（避免太复杂，结合正则，尽量清晰易懂）
    response.xpath('a/text()').re('Name\:(.*)')
    response.xpath('a/text()').re_first('Name\:(.*)')
  • node()选取所有类型节点，除了元素节点外还包含文本节点、注释节点等
    //div[@id="myid"]/following-sibling::node()[1]
  • following-sibling/preceding-sibling 同辈节点
    //div[@id="myid"]/preceding-sibling::node()[1]
  • 位置索引函数 position()
    //div[@id="myid"][position()<=3]
