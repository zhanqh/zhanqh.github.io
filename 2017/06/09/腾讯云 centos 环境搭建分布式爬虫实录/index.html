ssh 登录

chmod 400 <下载的与云服务器关联的私钥的绝对路径>
ssh -i “<下载的与云服务器关联的私钥的绝对路径>” @

如：ssh -i “~/workplace/ssh/macssh” root@123.207.40.174

python3 环境

编译环境准备

yum groupinstall ‘Development Tools’
yum install zlib-devel bzip2-devel openssl-devel ncurese-devel

下载、解压、编译安装pthon 3.5.2

wget https://www.python.org/ftp/python/3.5.2/Python-3.5.2.tar.xz
tar Jxvf Python-3.5.2.tar.xz
cd Python-3.5.2
./configure –prefix=/usr/local/python3
make && make install

修改系统环境

mv /usr/bin/python /usr/bin/python2.7
ln -s /usr/local/python3/bin/python3.5 /usr/bin/python -f
ln -s /usr/local/python3/bin/pip3 /usr/bin/pip -f

添加重新编译后的 python 的 bin 目录到系统 $PATH 中
vi ~/.bashrc ➜ 在文件末尾增加下面这行
export PATH=$PATH:/usr/local/python3/bin
然后执行
source ~/.bashrc

更新 yum 配置

vi /usr/bin/yum ➜ 修改第一行为：#!/usr/bin/python2.7

MongoDB

curl -O https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-3.4.0.tgz
tar -zxvf mongodb-linux-x86_64-3.4.0.tgz
mv mongodb-linux-x86_64-3.4.0/ /usr/local/mongodb
export PATH=/usr/local/mongodb/bin:$PATH

将 MongoDB 的 bin 目录到系统 $PATH 中
vi ~/.bashrc ➜ 在文件末尾增加下面这行
export PATH=$PATH:/usr/local/mongodb/bin
然后执行
source ~/.bashrc

Redis

wget http://download.redis.io/releases/redis-3.2.8.tar.gz
tar xvzf redis-3.2.8.tar.gz
cd redis-3.2.8/src
make

cp redis-server /usr/local/bin/
cp redis-cli /usr/local/bin/

mkdir /etc/redis
mkdir /var/redis
mkdir /var/redis/log
mkdir /var/redis/run
mkdir /var/redis/6379

cd ..
vi redis.conf 修改 redis.conf 配置使局域网内其他机器能访问：
bind 127.0.0.1 10.104.152.137
注意10.104.152.137为主节点 IP，即设置 redis 服务器要监听的地址
cp redis.conf /etc/redis/6379.conf

启动 redis 数据库
redis-server

ssh/scp 公私钥登录、文件传输

本地主机执行：
ssh-keygen -t rsa

将本地主机生成的公钥 /root/.ssh/id_rsa.pub 添加到远程主机的 /root/.ssh/authorized_keys 文件末尾

内网文件传输可执行：
scp root@10.186.4.96:/root/Python-3.5.2.tar.xz /root
即将远程主机 10.186.4.96 的 /root 目录下的 Python-3.5.2.tar.xz 文件传输到本地主机的 /root 目录下

指定密钥路径，使用参数 -i
scp -i "~/workplace/ssh/macssh" /root/Python-3.5.2.tar.xz root@10.186.4.96:/root
